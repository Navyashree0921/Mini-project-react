# 🌤️ WeatherWise

**WeatherWise** is a modern, responsive web application that provides real-time weather updates for cities globally. This project was developed to demonstrate proficiency in **asynchronous API handling**, **React Hooks**, and **dynamic UI rendering**.

---

## 🚀 Features

* **Live Data Retrieval:** Fetches real-time temperature, humidity, and wind speed using the **OpenWeatherMap API**.
* **Dynamic Backgrounds:** The UI theme and background change dynamically based on the current weather conditions (e.g., Clear, Clouds, Rain, Snow).
* **Global Search:** Instant weather lookups for thousands of cities worldwide with built-in error handling for invalid entries.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using **CSS**.

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Functional Components, Hooks like `useState` and `useEffect`)
* **Styling:** CSS 
* **API:** OpenWeatherMap API
---

## 📦 Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/WeatherWise.git](https://github.com/your-username/WeatherWise.git)
   cd WeatherWise
2. **Install Dependencies:**
   npm install
3. **Set up your API Key:**
   Create a .env file in the root directory and add your OpenWeatherMap API Key:
   REACT_APP_WEATHER_API_KEY=your_api_key_here
4. **Start the development server:**
   npm run dev
